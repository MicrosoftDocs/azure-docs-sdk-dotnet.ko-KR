# <a name="azure-mobile-libraries-for-net"></a>.NET용 Azure Mobile 라이브러리

## <a name="client-library"></a>클라이언트 라이브러리

Visual Studio [패키지 관리자 콘솔][PackageManager] 또는 [.NET Core CLI][DotNetCLI]를 사용하여 [NuGet 패키지](https://www.nuget.org/packages/Microsoft.Azure.Mobile.Server)를 직접 설치합니다.

#### <a name="visual-studio-package-manager"></a>Visual Studio 패키지 관리자

```powershell
Install-Package Microsoft.Azure.Mobile.Server
```

```bash
dotnet add package Microsoft.Azure.Mobile.Server
```
